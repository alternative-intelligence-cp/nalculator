# Nalculator

Nalculator is an advanced, native GTK4-based calculator application written in Nitpick. It is designed to provide Standard, Scientific, and Programmer modes, similar to modern OS calculators.

## Features

*   **Multi-Mode Calculation**: Switch seamlessly between Standard, Scientific, and Programmer interfaces.
*   **Native GUI**: Built on GTK4 for a fast, modern, and perfectly integrated Linux desktop experience.
*   **Graphing**: Advanced plotting and function visualization.
*   **Tape History**: Keep track of complex calculation sequences with history export capabilities.

## Building

To build Nalculator from source, ensure you have the `npkc` compiler (version `0.61.x` or later) and GTK4 development libraries installed.

```bash
npkc -picky src/main.npk
```

The resulting binary will be output by the build system.
