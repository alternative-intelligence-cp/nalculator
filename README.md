# Nalculator

Nalculator is an advanced, native GTK4-based calculator application written in Nitpick. It is designed to provide Standard, Scientific, and Programmer modes, similar to modern OS calculators.

## Features (Planned)
- Standard, Scientific, and Programmer modes
- Native GTK4 GUI
- Graphing Capabilities
- History & Tape Export
